# SherryAnalytics / Rain

A personal repo that contains my machine-learning coursework, a data diary, and side projects.

## Repo layout for SherryAnalytics/Rain/

- **`ml/`** — Course materials for my machine learning class: homework, project code, and class notes.
- **`diary/`** — Casual, chronological notes and writeups about things I learn and reflect day-to-day.
  - `diary/notes/` for short entries (dated filenames).
  - `diary/posts/` for longer, more polished writeups.
  - `diary/examples/` for runnable examples,
  - `diary/images/` for images.
- **`projects/`** — Longer-term and stand-alone projects (examples: wealth tracker, clustering experiments).
- **`README.md`** — This file: quick explanation and navigation.
